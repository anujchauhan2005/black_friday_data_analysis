# 🛒 Black Friday Data Analysis  

This repository contains a Jupyter Notebook that performs an analysis of Black Friday sales data using Python and popular data analysis libraries such as Pandas, Seaborn, and Matplotlib. The dataset used in this analysis is based on customer purchase behavior during Black Friday sales.

## 📌 Description  
This project focuses on analyzing customer purchase behavior during Black Friday sales. The goal is to uncover patterns, trends, and insights that can help businesses understand customer preferences and improve sales strategies.
